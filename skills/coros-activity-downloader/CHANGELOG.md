# COROS Activity Downloader 更新日志

## [2.0.0] - 2026-04-18

### 重大变更

- **迁移到 Python** - 主下载引擎从 PowerShell 迁移到 Python 3
  - 更好的 UTF-8 中文支持
  - 更清晰的代码结构
  - 跨平台兼容性

### 新增

- CLI 参数解析，支持灵活配置
- JSON 输出模式，便于程序化调用
- 智能重复检测（基于 labelId 提取）
- 文件完整性验证
- 详细的错误处理和日志

### 改进

- 支持任意命名约定的文件名
- 更好的错误提示
- 代码模块化重构
